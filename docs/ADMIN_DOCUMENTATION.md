# Admin Panel Documentation

This document provides an overview of the administrative functionalities available in the web application. The admin panel is designed to provide a centralized interface for managing the platform's content, users, and e-commerce operations.

## Accessing the Admin Panel

The admin panel is accessible to users with administrative privileges. To access it, navigate to `/account/admin` after logging in.

## Main Sections

The admin panel is organized into several main sections, each accessible from the sidebar navigation.

### 1. Dashboard

-   **Navigation Link**: `/account/admin`
-   **Purpose**: Provides a high-level overview of the platform's key metrics. This is the main landing page for the admin panel.

### 2. E-commerce

This section covers all aspects of the online store.

-   **Products**: `/account/admin/products`
    -   Manage all products available in the store.
    -   Add, edit, and delete products.
    -   Set product prices, descriptions, and images.
-   **Orders**: `/account/admin/orders`
    -   View and manage customer orders.
    -   Track order status and payment information.
-   **Inventory**: `/account/admin/inventory`
    -   Track product stock levels.
    -   Update inventory counts.
-   **Analytics**: `/account/admin/analytics`
    -   View sales reports and other e-commerce analytics.

### 3. Content Management

This section is for managing the website's content.

-   **Posts**: `/account/admin/posts`
    -   Create and manage blog posts.
-   **Works**: `/account/admin/works`
    -   Manage creative works, such as stories or articles.
-   **Timeline**: `/account/admin/timeline/events`
    -   Manage events on a historical timeline.
-   **Wiki**: `/account/admin/wiki`
    -   A comprehensive tool for managing the platform's wiki.
    -   **Wiki Manager**: Organize wiki content into a hierarchical structure of folders and pages.
    -   **Wiki Editor**: A rich text editor for creating and editing the content of wiki pages.
    -   **Category Manager**: Manage categories for organizing wiki pages.

### 4. User & System Management

-   **Users**: `/account/admin/users`
    -   View and manage all registered users.
-   **Beta Applications**: `/account/admin/beta-applications`
    -   Review and manage applications for the beta program.
    -   Approve or deny applications and provide feedback.

### 5. System & Configuration

-   **Webhooks**: `/account/admin/webhooks`
    -   Manage webhooks for integrating with third-party services.
-   **Media**: `/account/admin/media`
    -   Upload and manage media files, such as images and videos.
-   **Settings**: `/account/admin/settings`
    -   Configure various platform settings.
